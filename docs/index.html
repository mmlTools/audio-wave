What this plugin does

This plugin adds a new source type to OBS Studio: “Audio Wave (Simple)”. It listens to any audio source you choose (microphone, desktop audio, music player, etc.) and draws a clean visual that reacts to the sound.

You can render a classic horizontal waveform or wrap the audio around shapes like circles, rectangles, hexagons, stars, triangles, or diamonds using different styles such as lines, bars, or filled waves.

It is designed for creators who want a simple audio visualizer without complicated setup, scripting, or external programs.

Quick Note! Add a touch of style to your profile frame by dropping the wave or shape visualizer on top and linking it to your microphone or desktop audio source.

Quick start

  1. 1. Install the plugin

    Copy the plugin files into the correct OBS Studio obs-plugins folder for your operating system, then restart OBS Studio.

    • Windows:
      Place the plugin folder in:
      C:\Program Files\obs-studio\obs-plugins\64bit
      If you use the portable version, place it inside:
      obs-studio-portable\obs-plugins\64bit
    • macOS:
      Right-click OBS.app ‚Üí ‚ÄúShow Package Contents‚Äù, then copy the plugin into:
      OBS.app/Contents/PlugIns
    • Linux:
      System-wide installation:
      /usr/lib/obs-plugins
      or on some distros:
      /usr/lib/x86_64-linux-gnu/obs-plugins

      Flatpak version (user install):
      ~/.var/app/com.obsproject.Studio/config/obs-studio/plugins/

  2. 2. Add the “Audio Wave (Simple)” source

    In the Sources list of your scene, click the + button and choose Audio Wave (Simple) from the list.

  3. 3. Pick the audio source you want to visualize

    When the Properties window appears, use the Audio Source dropdown to choose which audio input you want the visualizer to react to. This can be your microphone, your game, your music, or any other active audio source in the scene.

  4. 4. Adjust the look to match your overlay

    Use the options in the Properties window to change Shape, Style, Wave Color, Width, Height, Amplitude (%), Curve Power (%), Shape Density (%), and Mirror wave horizontally until it fits your overlay style.

Controls explained (in plain language)

Audio Source

This tells the plugin which sound to listen to. Choose any existing source in your scene that has audio (like your microphone or desktop audio). If nothing moves, double-check that:

  • The chosen source is not muted.
  • There is actual sound on that source.

Shape & Style

These two options define the overall look of your visualizer.

  • Shape ‚Äì choose where the audio is drawn: Line (Horizontal), Rectangle, Circle, Hexagon, Star, Triangle, Diamond.
  • Style ‚Äì choose how it is drawn: Wave (Line), Wave (Bars), Wave (Linear Smooth), Wave (Linear Filled).

For a classic oscilloscope look, use Line + Wave (Line). For logo-style rings, try Circle + Wave (Bars) or Circle + Wave (Linear Filled).

Wave Color

This is the main color used to draw the line, bars, or filled wave. Pick a color that matches your overlay or brand.

Width & Height

These define the resolution of the visualizer in pixels. You can still scale the source directly in the preview like any other source, but:

  • Use a wider value (e.g. 800‚Äì1200) for a long bar under your scene.
  • Use a smaller value (e.g. 300‚Äì500) if you only need a small accent.

Amplitude (%)

Think of this as the visual sensitivity. Higher values make the wave or bars move more for the same audio level.

  • 100% ‚Äì 1:1 movement.
  • 200% ‚Äì more reactive.
  • 300‚Äì400% ‚Äì very bouncy for quiet audio.

Curve Power (%)

This adjusts how the visual reacts to quiet vs loud sounds. It shapes the response curve of the waveform.

  • Lower values ‚Äì more even reaction across the volume range.
  • Higher values ‚Äì quiet sounds stay subtle, loud sounds hit much harder visually.

Shape Density (%)

Controls how many bars/segments are drawn around shapes.

  • Lower values ‚Äì fewer, chunkier bars.
  • Higher values ‚Äì smoother, more detailed outline.

If performance is an issue, try lowering this value a bit, especially on complex shapes.

Mirror wave horizontally

When enabled, the plugin draws a second, mirrored copy of the wave or bars on the opposite side.

  • For the Line shape, it mirrors above and below the center line.
  • For shapes, it draws bars both inward and outward, giving a more symmetrical look.

Troubleshooting

The wave is completely flat. Nothing moves.
  • Check that you picked the correct Audio Source in the properties.
  • Make sure that source is not muted in the OBS mixer.
  • Play some audio or speak into the mic and look for levels in the mixer.
The visual moves, but it is too small or too big.

Use the Amplitude (%) slider. Lower it if the visual is too wild, or increase it if it hardly moves. You can also tweak Curve Power (%) to change how strongly loud parts of the audio are emphasized.

The wave looks pixelated when I scale it.

Increase the Width and Height values in the properties. Then resize the source in the preview to fit your layout.

The shape outline looks too sparse or too dense.

Adjust Shape Density (%). Lower values give fewer bars and a lighter look, higher values give a smoother, more continuous outline. If you notice performance drops, try reducing this value slightly.
